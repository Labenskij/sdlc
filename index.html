Инструкция и регламент

1. Общие положения

Система управления задачами предприятия «Черноморский молокозавод» предназначена для организации, отслеживания и контроля выполнения задач.

2. Роли и права доступа

  • Админ — полный доступ ко всем задачам
  • Руководство ЧЗПТ — просмотр всех задач и синхронизация из облака (без создания и редактирования)
  • Пользователь организации — видит только свои задачи; может создавать и редактировать свои

3. Облачная синхронизация

Доступна всем пользователям. Для работы требуется GitHub Personal Access Token с правами repo. Все пользователи работают с единым общим файлом tasks.json в централизованном репозитории.

4. Расчёт стоимости

Плановая = плановые часы × ставка. Фактическая = фактические часы × ставка.

5. Комментарии

К каждой задаче можно добавлять комментарии для обмена информацией между участниками.
