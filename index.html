Инструкция и регламент работы с системой

1. Общие положения

Система управления задачами предприятия «Черноморский молокозавод» предназначена для организации, отслеживания и контроля выполнения задач в рамках нескольких организаций. Система поддерживает облачную синхронизацию для резервного копирования и совместной работы.

2. Роли и права доступа

  • Администратор — полный доступ: просмотр, создание, редактирование и удаление всех задач, настройка синхронизации, экспорт/импорт данных
  • Пользователь организации — доступ только к своим задачам: создание, просмотр, редактирование и удаление записей, созданных под его учётной записью
  • Доступные организации: РАРУС, Градиент, Кутузовцы, Черноморский молокозавод, ЧЗПТ

3. Создание задачи

  • Нажмите кнопку «Новая задача»
  • Заполните обязательные поля: Название, Ответственный
  • Укажите приоритет, статус, плановые часы и ставку
  • Задача автоматически закрепляется за вашей учётной записью

4. Статусы задач

  • К выполнению — задача поставлена, ожидает начала работы
  • В работе — задача выполняется
  • На ревью — задача готова, ожидает проверки
  • Утверждено — задача проверена и одобрена
  • Заблокировано — выполнение задачи приостановлено
  • Завершено — задача полностью выполнена

5. Приоритеты задач

  • Высокий — критически важная задача
  • Средний — стандартная задача
  • Низкий — задача в свободное время

6. Облачная синхронизация (только для администратора)

  • Введите персональный ключ доступа
  • Целевой файл синхронизации фиксирован — tasks.json
  • «Синхронизация (Push)» — отправка данных в облако
  • «Загрузить из облака (Pull)» — получение данных из облака

7. График выполнения

Панель графика отображает распределение задач по статусам:

  • Для администратора — общая статистика по всем задачам
  • Для пользователей — статистика только по их задачам

8. Расчет стоимости

  • Плановая стоимость = Плановые часы × Ставка часа
  • Фактическая стоимость = Фактические часы × Ставка часа

9. Контакты

Кнопка «Контакты» в шапке содержит информацию по всем организациям и способам связи.

10. Поддержка

При возникновении вопросов обращайтесь к администратору системы предприятия «Черноморский молокозавод».
