Инструкция и регламент работы с системой

1. Общие положения

Система управления задачами предприятия «Черноморский молокозавод» предназначена для организации, отслеживания и контроля выполнения задач в рамках нескольких организаций. Система поддерживает синхронизацию с GitHub для резервного копирования и совместной работы.

2. Авторизация в системе

  • Для входа используйте учетные данные вашей организации
  • Доступные организации: РАРУС, Градиент, Кутузовцы, Черноморский молокозавод, ЧЗПТ
  • Администратор имеет доступ ко всем организациям

3. Создание задачи

  • Нажмите кнопку «Новая задача»
  • Заполните обязательные поля: Название, Организация, Ответственный
  • Укажите приоритет, статус, плановые часы и ставку
  • Нажмите «Сохранить задачу»

4. Статусы задач

  • К выполнению — задача поставлена, ожидает начала работы
  • В работе — задача выполняется
  • На ревью — задача готова, ожидает проверки
  • Утверждено — задача проверена и одобрена
  • Заблокировано — выполнение задачи приостановлено
  • Завершено — задача полностью выполнена

5. Приоритеты задач

  • Высокий — критически важная задача, требует немедленного выполнения
  • Средний — стандартная задача с обычными сроками
  • Низкий — задача может быть выполнена в свободное время

6. Синхронизация с GitHub

Синхронизация выполняется с репозиторием Labenskij/sdlc, файл tasks.json.

  • Введите GitHub Personal Access Token с правами repo
  • Целевой файл синхронизации фиксирован — tasks.json
  • Используйте «Синхронизация (Push)» для отправки данных в GitHub
  • Используйте «Загрузить из облака (Pull)» для получения данных из GitHub
  • При Push система автоматически определяет SHA файла для корректного обновления

7. Экспорт и импорт данных

  • Экспорт JSON — сохраняет все задачи в JSON файл
  • Загрузить JSON — импортирует задачи из JSON файла
  • Формат файла совместим с GitHub синхронизацией
  • Система автоматически нормализует ключи и значения при импорте (удаляет лишние пробелы)

8. Расчет стоимости

Стоимость задачи рассчитывается автоматически:

  • Плановая стоимость = Плановые часы × Ставка часа
  • Фактическая стоимость = Фактические часы × Ставка часа

9. Фильтрация задач

  • Используйте фильтры для отображения задач по организации
  • Фильтруйте задачи по статусу для быстрого доступа
  • Фильтры можно комбинировать

10. Редактирование и удаление

  • Нажмите «Редактировать» для изменения задачи
  • Нажмите «Удалить» для удаления задачи (требуется подтверждение)
  • Все изменения сохраняются автоматически в локальном хранилище

11. Рекомендации по работе

  • Регулярно синхронизируйте данные с GitHub для резервного копирования
  • Обновляйте статус задач по мере выполнения
  • Указывайте фактические часы для контроля затрат
  • Используйте приоритеты для планирования работы
  • Проводите ревью задач перед утверждением

12. Технические требования

  • Современный веб-браузер (Chrome, Firefox, Safari, Edge)
  • Стабильное интернет-соединение для синхронизации
  • Доступ к GitHub API для облачной синхронизации

13. Поддержка

При возникновении вопросов или проблем обращайтесь к администратору системы предприятия «Черноморский молокозавод».
